Development Timeline
Phase 1 2026년 초
기반 설계 — 세계관을 만들다
Vibe Studio의 첫 줄이 쓰인 순간. macOS 스타일 글라스모피즘 UI를 기반으로 독(Dock) 인터페이스, 라이트/다크 테마, 파티클 배경을 설계했습니다. 단 하나의 페이지에서 무한한 앱을 담을 수 있는 SPA 구조의 초석이 세워졌습니다.
SPA 아키텍처 테마 시스템 글라스모피즘 이메일 사전예약
Phase 2 2026년 2월 ~ 3월 초
핵심 앱 출시 — 생산성 도구의 탄생
생산성과 유틸리티를 위한 첫 번째 앱들이 연달아 탄생했습니다. PDF 분리, QR 생성·해독, 맞춤법 검사기, 박수 소음 측정기, 서버 모니터. "코딩 없이 만든다"는 개념이 실제 작동하는 도구로 증명된 단계입니다.
📄 PDF Splitter 🔗 QR Master ✍️ Spell Check 👏 Clap Meter 🖥️ Server Monitor
Phase 3 2026년 3월 10일 ~ 12일
게임 앱 통합 — 즐거움을 더하다
생산성만으로는 부족했습니다. 두 개의 게임이 Vibe Studio에 합류했습니다. 파친코는 중력 물리 시뮬레이션 기반의 미니 아케이드, 탱그램은 7개 조각으로 완성하는 두뇌 자극 퍼즐. 두 앱 모두 Canvas 기반으로 완전히 브라우저에서 실행됩니다. 탱그램 이식 과정에서 발생한 좌표계 문제를 해결하며 기술적 깊이도 높아졌습니다.
🎰 Pachinko 🧩 Tangram Canvas 물리엔진
Phase 4 2026년 3월 15일 ~ 17일
UI/UX 전면 리뉴얼 — 모바일을 품다
"PC에서만 잘 보이면 된다"는 생각을 버렸습니다. 모바일 터치 드래그 컨트롤, 햄버거 메뉴 드로어, 헤더/푸터 완전 통일. Vibe Studio는 이 단계를 거쳐 진정한 반응형 서비스로 거듭났습니다.
👆 터치 컨트롤 햄버거 드로어 반응형 완성
Phase 5 2026년 3월 18일
홈 리뉴얼 & SEO — 세상에 알리다
아무리 좋은 서비스도 찾지 못하면 의미 없습니다. 홈페이지 전면 리뉴얼과 함께 각 앱의 전용 랜딩 페이지를 제작했습니다. Schema.org 구조화 데이터, OG 태그, Canonical URL로 검색 엔진 최적화를 완성했고, 어드민 패널에 보안 이벤트 모니터링 기능을 추가했습니다.
SEO 최적화 7개 랜딩 페이지 보안 모니터링
Phase 6 2026년 3월 19일
코드 품질 — 지속 가능성을 설계하다
기능을 추가하는 것만큼 중요한 것은 유지보수 가능한 코드를 만드는 일입니다. 보안 자격증명 중앙화, 알림 UI 개선(팝업 → 토스트), 랜딩 페이지 템플릿 엔진 도입으로 7개 파일의 반복 코드를 설정 20줄로 압축했습니다. 자동화 배포 스크립트와 개발자 가이드도 이 단계에서 완성됐습니다.
보안 강화 토스트 알림 템플릿 엔진 배포 자동화
Phase 7 2026년 3월 19일
히스토리 공개 — 여정을 나누다
개발 과정을 외부와 공유하기 위한 공개 히스토리 페이지를 신설했습니다. Phase별 마일스톤, 앱 출시 연대기, 제작 철학을 담아 Vibe Studio의 이야기를 투명하게 공개했습니다. 내비게이션도 Home · About · History · Contact 4탭 구조로 확장됐습니다.
History 페이지 4탭 내비게이션 공개 아카이브
Phase 8 2026년 3월 23일 ~ 25일
Signal 출시 — AI 뉴스를 매일 전달하다
단순히 도구를 만드는 것을 넘어, 콘텐츠 서비스로 확장했습니다. Signal은 Gemini AI가 매일 수백 개의 AI 뉴스 중 신뢰도·중요도 기반으로 최대 8건을 선별해 한국어로 요약·게시하는 일간 AI 뉴스 서비스입니다. PHP + MariaDB 기반 파이프라인이 매일 오전 KST 기준으로 자동 수집·요약하며, 구독자에게는 헤드라인 이메일을 발송합니다. 구독에는 Google OAuth 원클릭 방식을 도입해 어느 페이지에서든 로그인한 계정이 자동으로 인식되는 전역 세션 구조를 구현했습니다. 내비게이션에 Signal 메뉴가 추가되어 서비스 전면에 노출됐습니다.
Signal 서비스 AI 뉴스 큐레이션 Google OAuth 구독 날짜별 아카이브 이메일 알림
Phase 9 2026년 3월 25일
후원 시스템 — 응원을 만나다
서비스를 좋아해주는 분들이 직접 응원을 표현할 수 있는 공간을 만들었습니다. PayApp PG 연동으로 실제 결제가 가능하고, 후원 완료 후 이름과 응원 메시지를 남기면 후원자 명단에 공개됩니다. 결제 상태 실시간 폴링, 이중 서버 검증, 취소 처리까지 완성된 결제 시스템입니다.
PayApp PG 후원자 명단 실시간 폴링 이중 검증
Phase 10 2026년 3월 26일
후원 관리 고도화 — 데이터 정합성을 완성하다
실제 후원 데이터를 운영하며 발견한 과제들을 해결했습니다. 어드민 패널에 후원 관리 탭을 신설해 KPI 대시보드, 상태 필터, 공개/비공개 토글, 수동 취소 기능을 갖췄습니다. 취소 웹훅 처리 강화, UTC→KST 시각 통일, 중복 DB 연결 제거로 후원 시스템이 안정적인 운영 체계를 갖추게 됐습니다.
후원 관리 탭 KST 시각 통일 데이터 정합성 코드 리팩토링

Design Philosophy
AI 바이브 코딩
코딩 지식 없이 아이디어만으로 앱을 만드는 새로운 방식. 기획자의 사고로 AI를 지휘합니다.
글라스모피즘
반투명 유리 질감의 UI로 깊이감과 프리미엄함을 동시에. macOS의 미학을 웹으로 재해석했습니다.
설치 없는 자유
앱 스토어 없이, 설치 없이, 브라우저 하나로. 모든 도구가 즉시 실행됩니다.
끊임없는 개선
출시가 끝이 아닙니다. 사용자 경험, 성능, 코드 품질 — 모든 것이 지속적으로 발전합니다.

이 페이지 자체도
AI와 함께 만들었습니다.

Vibe Studio는 계속 성장 중입니다 ✦
